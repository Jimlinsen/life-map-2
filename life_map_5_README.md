## 理论框架

### 1. 人生影响力系统（替代事件质量理论）

**核心公式**：
```
r(t) = r₀ + ∫v₀dt + Σ∫(G·Mⱼ/|rⱼ-rᵢ|²)dt

其中：
- r₀：出生位置
- v₀：八字决定的初始速度向量
- Mⱼ：第j个关系节点的质量
- |rⱼ-rᵢ|：与该节点的"距离"（关系亲密度的倒数）
```

**四大关系节点**：

| 柱位 | 关系 | 峰值影响 | 权重 | 映射宫位 |
|------|------|----------|------|----------|
| 年柱 | 祖上·父母 | 0-20岁 | 1.2 | 父母宫、福德宫 |
| 月柱 | 社会·事业 | 20-50岁 | 1.5 | 官禄宫、财帛宫、迁移宫 |
| 日柱 | 自我·配偶 | 终身 | 1.0 | 命宫、夫妻宫 |
| 时柱 | 子女·晚年 | 45-100岁 | 1.1 | 子女宫、田宅宫 |

### 2. 觉醒层级系统

| 层级 | 名称 | 与八字的关系 | 事件影响放大 |
|------|------|--------------|--------------|
| L1 | 执行层 | 被八字"决定" | +30% |
| L2 | 协调层 | 开始"观察"八字 | +10% |

- **提升觉醒层级 = 改变与八字的关系 = 实质性"改命"**

## 龙穴砂水升级版

### 龙：时间轨迹 + 觉醒演进

```
龙势类型 = 大运走势 × 觉醒状态

顺龙觉醒：大运相生 + 觉醒提升 → 最优轨迹
顺龙沉睡：大运相生但觉醒停滞 → 浪费先天优势
逆龙觉醒：大运相克但觉醒提升 → 逆境中成长
逆龙沉睡：大运相克 + 觉醒停滞 → 最需警醒
盘龙觉醒：大运循环 + 觉醒提升 → 螺旋上升
盘龙沉睡：大运循环 + 觉醒停滞 → 原地打转
```

### 穴：核心锚点 + 重要事件凝聚

```
穴位质量 = 星曜质量 + 事件锚定质量

真穴稳固：命宫强 + 正面事件多
真穴动荡：命宫强 + 负面事件多
虚穴借力：命宫弱 + 正面关系多
虚穴漂泊：命宫弱 + 关系薄弱
```

### 砂：关系力网络 + 人生影响力场

```
砂位结构：
├── 左砂_同辈：兄弟宫、仆役宫 + 朋友、同事、合作伙伴
├── 右砂_纵向：父母宫、子女宫 + 长辈、后辈、传承者
├── 前砂_发展：迁移宫、官禄宫 + 贵人、上级、客户
└── 后砂_稳定：田宅宫、福德宫 + 家人、根基、内心
```

### 水：能量流动 + 关系变迁

```
水口类型：
来水充沛：化禄强 + 新关系持续进入
去水过急：化忌强 + 关系频繁结束
聚水成潭：化权化科强 + 关系稳定深化
水枯源断：无化曜 + 关系僵化
```

## 核心洞见

### 1. 同八字不同命的解释

```
八字 = 进入四维时空的入口坐标 = 初始速度向量 v₀

命运轨迹 = v₀ + Σ(关系引力场积分)

不同的父母、配偶、贵人 = 不同的引力场
→ 导致完全不同的人生轨迹
```

### 2. 改命的三条路径

1. **改变关系引力场**：寻找贵人、优化关系网络
2. **提升觉醒层级**：降低事件对你的"质量感知"
3. **标记重要事件**：有意识地创造正向锚点

### 3. 理解即"减质"

```
事件质量 Q ∝ K(复杂度) × 2^{-D(理解深度)} × 觉醒系数

当你看清事件结构（D增加），遮蔽度降低
当你提升觉醒层级，感知到的事件质量减小
→ 事件对你的影响减弱
```
