# 人生地图 2.0 (Life Map)

基于紫微斗数与八字命理的可视化人生轨迹分析工具。输入出生信息，生成包含命盘、人生轨迹、关系引力场的完整人生地图。

## 核心功能

- **精确排盘** — 基于 `lunar-javascript` 八字排盘 + `iztro` 紫微斗数排盘
- **人生轨迹可视化** — 将大运、流年数据转化为可交互的折线图与雷达图
- **事件标记系统** — 标记人生重要事件（婚姻、事业、疾病等），计算其对轨迹的引力影响
- **关系引力场** — 标记重要关系（配偶、贵人、恩师），量化关系对人生走向的作用力
- **觉醒层级模型** — 5 级觉醒体系，衡量个体与命运的关系深度
- **龙穴砂水分析** — 将风水概念映射到人生维度，评估时运、锚点、人际与能量流
- **PDF 导出** — 一键生成完整人生地图报告

## 理论模型

```
人生轨迹 = 初始速度(八字) + ∫关系引力场dt

r(t) = r₀ + ∫v₀dt + Σ∫(G·Mⱼ/|rⱼ-rᵢ|²)dt
```

同八字不同命的核心解释：**相同的初始速度，不同的引力场**。

## 技术栈

| 层级 | 技术 |
|------|------|
| 框架 | React 18 + Vite 5 |
| 样式 | Tailwind CSS 3 |
| 图表 | Recharts |
| 命理 | iztro（紫微斗数）、lunar-javascript（八字） |
| 导出 | html2canvas + jsPDF |

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

访问 `http://localhost:5173` 即可使用。

## 使用方式

1. 输入出生年月日与时辰
2. 选择性别
3. 系统自动生成命盘与人生轨迹
4. 可选：添加重要事件与关系，观察引力场变化
5. 导出 PDF 报告
